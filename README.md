# Customer Shopping Behavior Analysis

This project analyzes customer shopping behavior using Python, SQL, and Power BI.  
The goal of this project is to understand customer purchasing patterns and generate insights that can help businesses make data-driven decisions.

## Tools & Technologies
- Python (Pandas)
- SQL
- Power BI
- Excel

## Project Steps

1. Data Preparation (Python)
- Loaded the dataset using Python
- Performed data cleaning and preprocessing
- Explored the dataset using Pandas

 2. Data Analysis (SQL)
- Imported the dataset into SQL database
- Used SQL queries to analyze customer behavior
- Identified trends such as spending patterns and category preferences

3. Data Visualization (Power BI)
- Built an interactive dashboard
- Visualized customer demographics and purchasing trends
- Highlighted key insights using charts and graphs

  Key Insights
- Identified customer age groups with highest spending
- Analyzed popular product categories
- Studied purchasing patterns across different customer segments

 Files in this Repository
- `customer_shopping_behavior.csv` – Dataset used for analysis
- `Customer_Shopping_Behavior_Analysis.ipynb` – Python analysis notebook
- `customer_behavior_sql_queries.sql` – SQL queries used for analysis
- `customer_behavior_dashboard.pbix` – Power BI dashboard

## Conclusion
This project demonstrates an end-to-end data analysis workflow including data cleaning, SQL analysis, and dashboard creation to generate meaningful business insights.
